ELO Token Demo is an open-source Web3 application built around the **ELO (Effortless Order)** token on the Binance Smart Chain. It provides a complete set of DeFi features — from buying tokens in a presale to minting NFTs and swapping tokens — all accessible through a React-based frontend with MetaMask and injected wallet support.

## Tech stack

| Layer | Technology |
|---|---|
| Frontend | React 17, Material-UI, Ant Design |
| Web3 | ethers.js, web3-react, Moralis |
| Backend | Node.js, Express |
| Database | MongoDB (Mongoose) |
| Blockchain | Solidity ^0.8.9, Truffle |
| Auth | JWT, bcrypt |

## Network

ELO Token Demo is deployed on **BSC Testnet**. See [deployed contract addresses](/reference/deployed-addresses) for the on-chain details.

<Steps>
  <Step title="Set up environment">
    Clone the repo and configure your `config.env` file with MongoDB and JWT settings
  </Step>
  <Step title="Install dependencies">
    Run `npm install` to install all frontend and backend packages
  </Step>
  <Step title="Start the app">
    Run `npm start` to launch both the Express backend and React dev server
  </Step>
  <Step title="Connect your wallet">
    Open `http://localhost:3000`, click **Connect Wallet**, and select MetaMask or another injected wallet
  </Step>
</Steps>
